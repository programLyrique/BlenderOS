unsigned int get_mem ( int  mem  ) 

Renseigne sur la quantité de mémoire disponible.

La mémoire basse commence à l'adresse 0, tandis que la mémoire haute commence à partir des 1 Mo La valeur maximale pour la mémoire basse est de 640 Ko. La valeur retournée pour la mémoire haute est au plus l'adresse du premier trou dans la mémoire, mais les 1 Mo de début. Il n'est pas garanti que ce soit cette valeur.

Il est donc conseillé d'utiliser la memory map pour une plus précise détection de la mémoire disponible

ParamÃ¨tres:
mem type de mémoire demandée :
  • MEM_LOWER pour la mémoire basse
  • MEM_UPPER pour la mémoire haute
Renvoie:
la quantité de mémoire désignée, 0 en cas d'erreur, le tout en ko
